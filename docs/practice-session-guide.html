3. Save & Manage Practice Plans

  • Click Save Plan to store your configuration
  • Saved plans show name, section count, total time
  • Load or delete plans as needed

4. Start & Run Practice Session

  • Review configuration and estimated total time
  • Click Start Session to begin
  • Practice timer (00:00), start/pause controls
  • Section navigation: progress indicator (Section 2 of 5), previous/next buttons
  • Current section highlighted on sheet music
  • Record practice notes for each section
  • Update confidence level after each section
  • Session auto-advances and ends after last section

5. Progress Tracking

  • Tracks time spent, confidence progression, notes, completion rates
  • Helps identify sections needing more work and track improvement

Best Practices

  • Mark specific measures, not whole pages
  • Focus on technical challenges and keep sections bite-sized
  • Start with short sessions (15-30 min), 3-5 min per section
  • Save multiple plans for different practice focuses
  • Update plans based on progress and effectiveness

Practice Workflow Example

  1. Morning Setup (5 min): Load plan, review sections, start session
  2. Focused Practice (20 min): Work through sections, take notes, update confidence
  3. Session Review (5 min): Review notes, plan next session, save updates

Getting Started

  • Try a simple session: mark 2-3 short sections, practice for 10 min
  • Experiment with methods and build routine
  • Track progress with confidence updates

Consistent, focused practice with clear goals is more effective than long, unfocused sessions!
